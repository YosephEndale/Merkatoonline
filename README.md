# Merkatoonline - Django E-Commerce

## Quick Start

### Run the Server
```bash
cd /home/yoseph-endale/Projects/Merkatoonline
source .venv/bin/activate
python manage.py runserver 0.0.0.0:8000
```

Visit: **http://localhost:8000**

### Directories
- `templates/` - HTML pages
- `static/` - CSS, JavaScript, Images (including logo)
- `store/` - Products & shopping
- `users/` - Authentication & profiles
- `merkatoonline/` - Django settings

### Admin Panel
http://localhost:8000/admin/
